Repara Ya

Soluciones cercanas, seguras y confiables

Pregunta Problema

¿Cómo a través de una aplicación podemos mejorar la comunicación entre locales y clientes para encontrar sitios donde se puedan arreglar dispositivos electrónicos?

Objetivo Principal

En ocasiones, los clientes tienen dificultad para encontrar servicios confiables y cercanos para reparar sus dispositivos electrónicos y electrodomésticos. La falta de información y la desconfianza en la honestidad y la calidad de los servicios generan confusión y frustración en los usuarios. Repara Ya es una aplicación que pretende brindar una mayor accesibilidad a la hora de buscar una solución a los equipos electrodomésticos, otorgando información de un local o un técnico cercano, así brindando la mejor opción.

Gestión de Base de Datos

Objetivos del Proyecto

  1. Identificar y resolver problemas reales en la app usando programación y herramientas como UML, prototipos y mockups para diseñar soluciones prácticas.
  2. Aprender a crear y gestionar una base de datos con MySQL y Workbench para organizar bien la información de la app.
  3. Usar lo aprendido para resolver problemas paso a paso con lógica, creando y probando funciones en Python para la app de reparación.
  4. Crear una función en la app que asigne al técnico más cercano al cliente usando ubicaciones y un algoritmo de distancia.
  5. Aprender a estructurar y documentar un proyecto tecnológico, desde el problema hasta su desarrollo.
